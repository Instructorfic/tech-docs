# Lógica de Programación y Pensamiento Computacional

## Descripción

Asignatura orientada al desarrollo del razonamiento lógico y la resolución estructurada de problemas mediante algoritmos.

---

## Competencias

Al finalizar el curso el estudiante podrá:

- Diseñar algoritmos claros y estructurados.
- Aplicar estructuras de control.
- Utilizar diagramas de flujo.
- Resolver problemas de forma sistemática.
