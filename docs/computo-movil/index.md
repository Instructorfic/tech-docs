# Cómputo Móvil

## Descripción

Asignatura orientada al desarrollo de aplicaciones móviles modernas, abordando arquitectura, ciclo de vida, interfaces y buenas prácticas.

---

## Competencias

Al finalizar el curso el estudiante será capaz de:

- Comprender la arquitectura de aplicaciones móviles.
- Implementar interfaces funcionales.
- Gestionar estados y ciclo de vida.
- Aplicar principios de diseño y buenas prácticas.
