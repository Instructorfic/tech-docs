# Desarrollo, seguridad y operaciones de sistemas

## Descripción

Curso enfocado en la integración de seguridad dentro del ciclo de vida del desarrollo de software, automatización, CI/CD y buenas prácticas.

---

## Competencias

El estudiante será capaz de:

- Comprender el enfoque DevSecOps.
- Implementar pipelines seguros.
- Identificar vulnerabilidades comunes.
- Aplicar controles de seguridad automatizados.
